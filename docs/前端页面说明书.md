## 2. 页面结构

### 2.1 登录页 (Login.vue)

#### 2.1.1 页面描述

用户登录系统页面，提供用户名和密码登录功能。

#### 2.1.2 页面功能

- 用户名输入
- 密码输入（支持显示/隐藏）
- 登录按钮
- 记住密码功能
- 登录表单验证

#### 2.1.3 页面布局

- 居中卡片式布局
- 顶部显示系统Logo和名称
- 中部为登录表单
- 底部显示版权信息

#### 2.1.4 接口调用

- POST /api/v1/auth/login - 登录接口
- POST /api/v1/auth/me - 获取当前用户信息

#### 2.1.5 路由配置

```typescript
{
  path: '/login',
  name: 'Login',
  component: () => import('@/views/Login.vue'),
  meta: { title: '登录', requiresAuth: false }
}
```

### 2.3 仪表盘 (Dashboard.vue)

#### 2.3.1 页面描述

系统首页，展示关键业务指标和数据图表。

#### 2.3.2 页面功能

- 显示核心业务指标卡片
- 显示采购趋势图表
- 显示销售趋势图表
- 显示低库存预警列表
- 显示待处理订单列表

#### 2.3.3 页面布局

- 顶部：4个指标卡片（产品总数、供应商数、客户数、待处理订单）
- 中部：2个图表区域（采购趋势、销售趋势）
- 底部：2个列表区域（低库存预警、待处理订单）

#### 2.3.4 接口调用

- GET /api/v1/reports/dashboard - 获取仪表盘数据
- GET /api/v1/reports/inventory-status - 获取库存状态
- GET /api/v1/reports/purchase-summary - 获取采购汇总
- GET /api/v1/reports/sales-summary - 获取销售汇总

## 3. 系统管理模块

### 3.1 用户管理 (UserManagement.vue)

#### 3.1.1 页面描述

管理系统用户，支持用户的增删改查。

#### 3.1.2 页面功能

- 用户列表展示（表格形式）
- 用户搜索（用户名、真实姓名、部门）
- 新增用户
- 编辑用户
- 删除用户
- 重置密码
- 启用/禁用用户

#### 3.1.3 表格字段

| 字段 | 说明 |
|------|------|
| ID | 用户ID |
| 用户名 | 登录用户名 |
| 真实姓名 | 用户真实姓名 |
| 邮箱 | 用户邮箱 |
| 手机号 | 用户手机号 |
| 部门 | 所属部门 |
| 状态 | 启用/禁用 |
| 创建时间 | 用户创建时间 |
| 操作 | 编辑、删除、重置密码 |

#### 3.1.4 表单字段

新增/编辑用户表单：
- 用户名（必填）
- 密码（新增时必填）
- 真实姓名（必填）
- 邮箱
- 手机号
- 所属部门
- 状态

#### 3.1.5 接口调用

- GET /api/v1/users - 获取用户列表
- POST /api/v1/users - 创建用户
- PUT /api/v1/users/{id} - 更新用户
- DELETE /api/v1/users/{id} - 删除用户
- POST /api/v1/users/{id}/reset-password - 重置密码

## 8. 路由配置

### 8.1 路由结构

```typescript
const routes = [
  {
    path: '/login',
    name: 'Login',
    component: () => import('@/views/Login.vue'),
    meta: { requiresAuth: false }
  },
  {
    path: '/',
    component: () => import('@/views/Layout.vue'),
    meta: { requiresAuth: true },
    children: [
      {
        path: '',
        name: 'Dashboard',
        component: () => import('@/views/Dashboard.vue'),
        meta: { title: '仪表盘' }
      },
      // 系统管理路由
      {
        path: 'system',
        children: [
          { path: 'users', component: () => import('@/views/system/Users.vue') },
          { path: 'roles', component: () => import('@/views/system/Roles.vue') },
          { path: 'departments', component: () => import('@/views/system/Departments.vue') },
          { path: 'menus', component: () => import('@/views/system/Menus.vue') }
        ]
      },
      // 供应链管理路由
      {
        path: 'supply',
        children: [
          { path: 'suppliers', component: () => import('@/views/supply/Suppliers.vue') },
          { path: 'procurement', component: () => import('@/views/supply/Procurement.vue') },
          { path: 'inventory', component: () => import('@/views/supply/Inventory.vue') },
          { path: 'sales', component: () => import('@/views/supply/Sales.vue') }
        ]
      },
      // 财务管理路由
      {
        path: 'finance',
        children: [
          { path: 'payments', component: () => import('@/views/finance/Payments.vue') },
          { path: 'bills', component: () => import('@/views/finance/Bills.vue') },
          { path: 'accounts', component: () => import('@/views/finance/Accounts.vue') },
          { path: 'reports', component: () => import('@/views/finance/Reports.vue') }
        ]
      }
    ]
  }
]
```

### 8.2 路由守卫

**认证守卫：**
- 检查用户是否登录
- 未登录跳转到登录页

**权限守卫：**
- 检查用户是否有权限访问
- 无权限提示无权访问
