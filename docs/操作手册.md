## 1. 系统登录

### 1.1 访问系统

1. 打开浏览器，输入系统地址：http://localhost:5173
2. 进入登录页面

### 1.2 登录系统

1. 在用户名输入框中输入用户名
2. 在密码输入框中输入密码
3. 点击"登录"按钮

**默认管理员账号：**
- 用户名：admin
- 密码：admin123

### 1.3 修改密码

登录后建议立即修改默认密码：

1. 点击右上角用户头像
2. 选择"修改密码"
3. 输入原密码和新密码
4. 点击"确认"按钮

### 1.4 退出系统

点击右上角用户头像，选择"退出登录"

## 2. 系统管理

### 2.1 用户管理

#### 2.1.1 查看用户列表

1. 点击左侧菜单"系统管理" → "用户管理"
2. 系统显示用户列表，包含用户名、真实姓名、邮箱、手机号、部门、状态等信息

#### 2.1.2 添加用户

1. 点击"添加用户"按钮
2. 填写用户信息：
   - 用户名（必填）
   - 密码（必填）
   - 真实姓名
   - 邮箱
   - 手机号
   - 所属部门
   - 状态
3. 点击"确认"按钮保存

#### 2.1.3 编辑用户

1. 在用户列表中找到要编辑的用户
2. 点击操作列的"编辑"按钮
3. 修改用户信息
4. 点击"确认"按钮保存

#### 2.1.4 删除用户

1. 在用户列表中找到要删除的用户
2. 点击操作列的"删除"按钮
3. 确认删除操作

#### 2.1.5 重置密码

1. 在用户列表中找到要重置密码的用户
2. 点击操作列的"重置密码"按钮
3. 输入新密码
4. 点击"确认"按钮保存

## 6. 常见问题

### 6.1 登录问题

**Q: 无法登录系统？**

A: 请检查：
- 用户名和密码是否正确
- 用户状态是否启用
- 网络连接是否正常

### 6.2 数据问题

**Q: 保存数据时提示错误？**

A: 请检查：
- 必填字段是否填写完整
- 数据格式是否正确
- 是否存在重复数据

### 6.3 权限问题

**Q: 某些功能无法访问？**

A: 请检查：
- 用户是否具有相应权限
- 角色权限是否正确配置

### 6.4 性能问题

**Q: 系统响应较慢？**

A: 请检查：
- 网络连接是否正常
- 数据量是否过大
- 是否需要清理缓存

### 7.1 数据安全

- 定期修改密码
